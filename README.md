# CWC2023-Analysis

The ICC has gathered comprehensive data from the 2023 Cricket World Cup, including batting, bowling, fielding, and venue stats. As a data analyst, the task is to perform exploratory data analysis (EDA) on this dataset.

Goal of this project is to uncover insights on player performances, scoring patterns at different venues, and identify the top 10 batsmen and bowlers of the tournament.

# Objective

Load and Explore the Data
Perform Statistical and Exploratory Analysis
Clean and Transform the Data
Visualize the Data
Generate Insights
